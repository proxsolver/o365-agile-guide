Case 07: 안전 점검 체크리스트 앱

Microsoft Forms로 일일 안전 점검 디지털화, 이상 발견 시 즉시 알림

Case 07 | 안전관리

🛡️ 안전 점검 체크리스트 앱

종이 체크리스트를 Forms로 전환하고, 이상 항목 발견 시 자동 알림으로 즉각 대응합니다

디지털 점검
즉시 알림
이력 분석

문제 상황

현장의 목소리

안전담당 한OO 주임이 ISO 45001 심사 준비 중 문제를 발견합니다:
"매일 아침 작업 전 안전 점검을 하는데, 종이 체크리스트에 체크만 하고 끝이에요. 이상 발견해도 보고가 늦어지고, 과거 점검 기록 찾으려면 창고에서 서류함을 뒤져야 합니다. 심사관이 '3개월 전 점검 기록'을 요청하면 찾는 데만 30분이 걸려요. 그리고 점검을 했는지 안 했는지 실시간 확인도 안 됩니다."

일일 점검 횟수3회/라인
월간 기록180건
개선 기간2주

• B동 창고
• 옥외 야적장

# 점검 항목 유형 선택지

9 바닥 상태 (미끄럼 위험) 선택 ✅ 정상 / ⚠️ 이상 / ➖ 해당없음

🔹 이상 발견 시 상세 (2개)
# 질문 유형 설정
14 이상 항목 상세 설명 텍스트 긴 답변, 조건부 (이상 선택 시만 표시)
15 현장 사진 파일 업로드 이미지만, 최대 3개
1-3
질문 추가하기 상세 단계
📍 "선택" 질문 만들기:

Step 1: "+ 새로 추가" 클릭
Step 2: "선택" 아이콘 클릭
Step 3: 질문 입력: "비상구 통로 확보"
Step 4: 옵션 입력:
옵션 1: ✅ 정상
옵션 2: ⚠️ 이상
옵션 3: ➖ 해당없음
Step 5: "필수" 토글 켜기
Step 6: 우측 "..." → "드롭다운" 선택 (공간 절약)

💡 Tip: 첫 번째 질문 만든 후 "복제" 버튼으로 빠르게 복사!
1-4
조건부 질문 설정 (분기)
💡 "이상" 선택 시에만 상세 설명 표시하기:

Step 1: "이상 항목 상세 설명" 질문 선택
Step 2: 우측 "..." → "분기 추가"
Step 3: 분기 조건 설정:
IF 질문 4~13 중 하나라도 = "⚠️ 이상"
THEN 이 질문으로 이동
⚠️ 주의: Forms 분기는 단순해서, 복잡한 조건은 Power Automate로 처리
1-5
파일 업로드 질문 설정
📍 현장 사진 업로드 설정:

Step 1: "+ 새로 추가" → "파일 업로드" 선택
Step 2: 질문: "현장 사진 (이상 발견 시)"
Step 3: 파일 제한 설정:
  • 파일 수 제한: 3개
  • 단일 파일 크기: 10MB
  • 파일 형식: 이미지만 (jpg, png)

📁 업로드된 파일은 OneDrive > Apps > Microsoft Forms 폴더에 저장됩니다.
1-6
Forms 설정 최적화
📍 설정 (우측 상단 "..." → "설정"):

응답 옵션:

☑️ 응답자에게 결과 표시 → 끄기 (결과 비공개)
☑️ 진행률 표시줄 → 끄기

알림:
☑️ 각 응답에 대한 이메일 알림 받기 → 켜기

응답자 기록:
☑️ 이름 기록 → 켜기 (자동으로 로그인 사용자 기록)

❓ 자주 묻는 질문 (FAQ)

Q1. Forms에서 사진 업로드가 안 돼요

A: 파일 업로드는 조직 내 사용자만 가능합니다.

확인: Forms 설정 → "누가 이 양식을 채울 수 있나요?" → "내 조직의 사용자만" 선택

외부인도 사용해야 한다면 파일 업로드 대신 "링크 입력" 질문으로 대체

Q2. 점검 항목을 추가하고 싶어요

A: Forms에서 언제든 질문 추가 가능합니다.

주의: Power Automate 흐름에서 새 필드를 매핑해야 합니다.
1. Forms에 질문 추가
2. Power Automate 편집 → 조건/저장 로직에 새 필드 추가

Q3. 오프라인에서도 점검할 수 있나요?

A: Forms는 온라인 전용입니다.

대안:
• Power Apps로 오프라인 지원 앱 개발 (고급)
• 현장 Wi-Fi 확장으로 연결 보장
