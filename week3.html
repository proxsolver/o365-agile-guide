🚀 스프린트 실행

Daily Standup과 함께 첫 번째 스프린트를 실행합니다

📋 단계별 실행 가이드

1
Teams
10분/일

Teams 반복 회의 설정 (Daily Standup)

왜 필요한가요?

Daily Standup은 팀원 간 동기화를 위한 짧은 미팅입니다. 장애물을 빠르게 발견하고 해결합니다.

실행 순서

1-1
반복 미팅 예약

Teams → 캘린더 → "새 모임" → 반복: 매일

  • 제목: Daily Standup
  • 시간: 오전 9:30 (15분)
  • 채널: Daily Standup 채널
1-2
미팅 진행 방식

각 팀원이 3가지 질문에 답합니다:

  • 🟢 어제 무엇을 완료했나요?
  • 🔵 오늘 무엇을 할 예정인가요?
  • 🔴 장애물이 있나요?

완료 체크리스트

2
Planner
수시

Planner 차트로 진행 현황 모니터링

2-1
태스크 상태 업데이트

작업 시작 시: To Do → In Progress로 드래그
작업 완료 시: In Progress → Done으로 드래그

2-2
차트 뷰 확인

상단 "차트" 탭 클릭하여 진행 상황 시각화

  • 상태별 작업 수
  • 담당자별 작업 분포
  • 우선순위별 현황
3
OneNote
5분/일

OneNote에 회의록/의사결정 기록

3-1
Daily Standup 기록

Daily Standup 섹션에 매일 페이지 생성

📅 2025-01-21 Daily Standup
---
홍길동:

- 오늘: API 구현 시작

3-2
의사결정 기록

중요한 기술적/비즈니스 결정은 별도 페이지에 기록
